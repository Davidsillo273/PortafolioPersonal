Mis conocimientos

Siendo un FullDeveloper, logro, manejo y sé crear FrontEnds, BackEnds, bases de datos relacionales y no relacionales, y lenguajes de programación de vanguardia en la actualidad.


Mis Lenguajes de Programación


JavaScript

Java

Java es un lenguaje de programación de alto nivel, orientado a objetos y multiplataforma, diseñado para ser rápido, seguro y confiable. Desarrollado por Sun Microsystems (ahora Oracle) en 1995, permite ejecutar el mismo código en distintas plataformas (desktops, móviles, servidores) gracias a la Máquina Virtual Java (JVM).

C#

C# (C Sharp) es un lenguaje de programación moderno, orientado a objetos y de tipado fuerte, desarrollado por Microsoft en el año 2000 y que funciona sobre la plataforma .NET. Es conocido por ser multiplataforma, seguro y eficiente, ideal para desarrollar aplicaciones web, móviles, de escritorio, juegos y servicios en la nube.


Mis Lenguajes SQL


SqlServer

Microsoft SQL Server es un sistema de gestión de base de datos relacional, desarrollado por la empresa Microsoft. El lenguaje de desarrollo utilizado es Transact-SQL, una implementación del estándar ANSI del lenguaje SQL, utilizado para manipular y recuperar datos, crear tablas y definir relaciones entre ellas.

SQLDeveloper

Oracle SQL Developer es un entorno de desarrollo integrado (IDE) gratuito y gráfico, basado en Java, que simplifica la creación, gestión y depuración de bases de datos Oracle, tanto tradicionales como en la nube. Permite ejecutar consultas SQL, desarrollar scripts PL/SQL, modelar datos y realizar tareas de administración de bases de datos (DBA).


Mis Lenguajes NoSQL


MongoDB

MongoDB es una base de datos NoSQL de código abierto, orientada a documentos, diseñada para la flexibilidad y escalabilidad en aplicaciones modernas. Almacena datos en estructuras tipo JSON (llamadas BSON), evitando tablas y filas rígidas, lo que facilita el manejo de información no estructurada, consultas rápidas y escalado horizontal.
